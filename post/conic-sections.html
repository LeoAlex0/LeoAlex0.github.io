Conic Sections: 圆锥曲线、焦点与离心率

数学, 几何 圆锥曲线, GeoGebra

圆锥曲线来自一个朴素的动作：用一个平面去切一个双圆锥。平面角度不同，截痕可能是椭圆、抛物线或双曲线；如果退化，还会得到点、直线或一对相交直线。

更适合计算的说法是：圆锥曲线是二次方程

\[ Ax^2+Bxy+Cy^2+Dx+Ey+F=0 \]

在平面上的轨迹。只看二次项时，判别式

\[ \Delta = B^2 - 4AC \]

给出一个快速分类：

条件 曲线类型
\(\Delta < 0\) 椭圆，圆是特殊情形
\(\Delta = 0\) 抛物线
\(\Delta > 0\) 双曲线

焦点定义

还有一个统一的几何定义：点 \(P\) 到焦点 \(F\) 的距离，和它到准线 \(l\) 的距离保持固定比例：

\[ \frac{|PF|}{d(P,l)} = e \]

这里 \(e\) 叫做离心率。它像一个调节旋钮：

  • \(0 < e < 1\)：椭圆。
  • \(e = 1\)：抛物线。
  • \(e > 1\)：双曲线。

如果把离心率想成“偏离圆的程度”，圆就是 \(e=0\) 的极端情况；椭圆偏一点，抛物线刚好打开，双曲线则彻底分成两支。

椭圆

以原点为中心、长轴在 \(x\) 轴上的椭圆写作

\[ \frac{x^2}{a^2}+\frac{y^2}{b^2}=1,\qquad a>b>0 \]

焦点为 \((\pm c,0)\)，其中

\[ c^2=a^2-b^2,\qquad e=\frac{c}{a} \]

椭圆最漂亮的性质是“焦点距离和为常数”：曲线上任意一点 \(P\) 都满足

\[ |PF_1| + |PF_2| = 2a \]

下面这段 Python 可以生成椭圆上的采样点。它主要用于确认代码高亮是否正常工作：

from math import cos, pi, sin


def ellipse_points(a, b, count=12):
    for index in range(count):
        theta = 2 * pi * index / count
        yield a * cos(theta), b * sin(theta)


for x, y in ellipse_points(a=4.2, b=2.8, count=6):
    print(f"({x: .2f}, {y: .2f})")

抛物线

抛物线是离心率 \(e=1\) 的边界情形：点到焦点和准线的距离相等。最常见的形式是

\[ y^2=4px \]

其中焦点是 \((p,0)\)，准线是 \(x=-p\)。如果开口向上，也常写作

\[ x^2=4py \]

双曲线

双曲线可以看成椭圆的“距离和”变成了“距离差”。标准式

\[ \frac{x^2}{a^2}-\frac{y^2}{b^2}=1 \]

对应焦点 \((\pm c,0)\)，其中

\[ c^2=a^2+b^2,\qquad e=\frac{c}{a}>1 \]

曲线上任意一点 \(P\) 满足

\[ \bigl||PF_1|-|PF_2|\bigr| = 2a \]

圆锥曲线有一种很适合作为博客演示的气质：公式能说明结构，代码能生成样本，GeoGebra 则让读者直接“摸到”不变量。三者放在一起，静态文章也会活起来。
